# 오늘의 라이딩
비트캠프 169기 파이널 프로젝트 (team. hongdukae)<br>
라이딩을 취미로 하는 사람들에게 다양한 코스를 추천해주고 라이더들을 위한 커뮤니티를 활성화한 웹 페이지

## 사용기술
- Oracle
- Tomcat
- MyBatis
- SpringBoot
- java
- HTML & CSS
- javascript
- JQuery
- Ajax
- BootStrap

## 담당한 기능
1. meeting Board
  + Ajax을 이용하여 게시판 목록 동적 노드로 불러오기
  + 페이징 처리
  + 상세페이지 불러오기
  + 게시글 수정, 삭제
2. meeting, review Board UI 담당
  + bootstrap 사용
  + layout 및 아이콘 선정
